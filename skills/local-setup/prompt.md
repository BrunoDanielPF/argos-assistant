# Local Setup

Use this skill when setting up Argos locally.

Workflow:
- Confirm Python version and current working directory.
- Install the package in editable mode with dev dependencies when developing.
- Verify Ollama daemon availability at the configured endpoint.
- Pull or list the configured model.
- Run CLI help, tests, and one smoke prompt.

Output:
- Setup commands.
- Expected outputs.
- Troubleshooting for missing PATH, stopped Ollama, missing model, and timeout.
